🔄 Hỏi AI báo cáo nhanh
🔑 Gemini API Key
Lấy miễn phí tại aistudio.google.com → Get API Key
Dùng tài khoản Gemini Pro của bạn (miễn phí đến 9/2026)
📊 Google Sheets
Xem hướng dẫn tạo Apps Script trong file Code.gs kèm theo.
Deploy → Web App → Anyone → Copy URL dán vào đây.
Lấy từ URL Google Sheets: .../d/[ID này]/edit
📱 Telegram Bot
Tạo bot tại @BotFather trên Telegram → /newbot
Gửi tin nhắn cho bot → mở
api.telegram.org/bot[TOKEN]/getUpdates
Lấy chat.id trong kết quả
⚙️ Tùy chọn
📖 Hướng dẫn nói chuyện với AI
💰 Thu tiền:

"Khách Nguyễn An trả tiếp 3 triệu đơn sofa"

"Nhập hàng [tên hàng] từ [NCC], giá [số tiền], vận chuyển [tên người] [số tiền]"
"Chi mua sản phẩm A của Công ty B 7 triệu"
